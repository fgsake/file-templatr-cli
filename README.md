# File Templatr CLI

```
Usage: file-templatr [options] <TargetName>

Simple CLI Utility for creating new files and folder based on a templated folder/file structure

Arguments:
  TargetName                 Target Name for newly created template

Options:
  -V, --version              output the version number
  -p, --path [path]          Destination folder for the component (default: current directory)
  -t, --template <template>  Template you want to use from your `.fs-templates` directory
  -f, --force                Force overwrite any files
  -d, --debug                Enable debug mode
  -h, --help                 display help for command
```
